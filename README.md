# 炖时光微信小程序

这是 `炖时光` 的微信小程序项目骨架，目标是把门店的下单、预订、库存、会员、充值和数据分析整合到一个轻量、好维护的系统里。

## 已包含

- 顾客端：首页、菜单、组合套餐、订单、会员、门店
- 店长端：库存确认、订单管理、日报汇总、数据看板
- 真实门店路线图
- 真实菜品图片素材
- 开业 88 折与限时折扣的基础逻辑
- 组合套餐自动计价

## 本地打开方式

1. 用微信开发者工具导入本仓库根目录。
2. 将项目根目录识别为 `miniprogram/`。
3. 打开首页即可预览完整骨架。

## 说明

- 当前版本是前端骨架和静态数据版本，方便先确认页面与流程。
- 后续可以继续接入支付、订单接口、后台数据库和消息通知。
